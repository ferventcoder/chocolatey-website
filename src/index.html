The package manager for Windows

The sane way to manage software on Windows

  • Chocolatey can easily manage all aspects of Windows software whether installation, configuration, upgrade, or uninstallation.
  • Works with binaries and existing installer technologies like MSI, NSIS, InnoSetup, etc.
  • Chocolatey can download resources from file shares, http, and ftp locations, but it works best when the software a package represents is embedded in the package files.
  • Optionally use PowerShell to provide automated software management instructions and Chocolatey’s module with awesome functions that really speed up the packaging process!

Trusted by Business

  • Businesses are building and hosting their own packages that are reliable and repeatable so they can fully trust and control the process.
  • Businesses trusting Chocolatey for production purposes do not use the packages found on this site (the community package repository). Find out why.
  • When businesses need more features than they can find in the Open Source version of Chocolatey, they turn to [Chocolatey for Business](link to Business page).

Official Chocolatey tools

  • The CLI client, choco.exe, was designed with security in mind. Know you are using Chocolatey from a trusted source when you are using the official Chocolatey clients.
  • Use choco.exe to install, upgrade, uninstall, publish and search for packages (and much more).
  • choco.exe has tab completion and excellent inline help.
  • Use [Chocolatey GUI](link to GUI package) when you need a visual representation of packages.

Packages are independent deployment artifacts

  • Chocolatey packages are not just fancy zip files, they understand versioning and dependencies, allowing you to encapsulate everything related to managing installation and configuration for a piece of software.
  • As an independent deployment artifact, Chocolatey packages are not tied to another technology, allowing developers to install packages with Chocolatey and System Administrators to deploy Chocolatey packages using Chocolatey’s integration with their favorite configuration manager like SCCM, Puppet, or Chef.
  • Chocolatey packages allow you to separate software installation concerns from automation tools / configuration managers and use the best tool for software management, which is the Chocolatey framework.

Choose from thousands of community packages

  • Chocolatey.org has the largest online registry of Windows packages.
  • Package submissions go through a rigorous moderation review process.
  • The packages in the community repository are automatically scanned for viruses.
  • The community repository has a strict policy on malicious and pirated software.
